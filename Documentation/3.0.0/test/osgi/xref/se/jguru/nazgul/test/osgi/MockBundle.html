123      * Starts this bundle.
124      * If this bundle's state is {@code UNINSTALLED} then an
125      * {@code IllegalStateException} is thrown.
126      * If the current start level is less than this bundle's start level:
127      * <ul>
128      * <li>If the {@link #START_TRANSIENT} option is set, then a
129      * {@code BundleException} is thrown indicating this bundle cannot be
130      * started due to the Framework's current start level.
131      ´* <li>Otherise, the Framework must set this bundle's persistent autostart
132      * setting to <em>Started with declared activation</em> if the
133      * {@link #START_ACTIVATION_POLICY} option is set or
134      * <em>Started with eager activation</em> if not set.
135      * </ul>
136      * When the Framework's current start level becomes equal to or more than
137      * this bundle's start level, this bundle will be started.
138      * Otherwise, the following steps are required to start this bundle:
139      * <ol>
140      * <li>If this bundle is in the process of being activated or deactivated
141      * then this method must wait for activation or deactivation to complete
142      * before continuing. If this does not occur in a reasonable time, a
143      * {@code BundleException} is thrown to indicate this bundle was unable to
144      * be started.
145      * <li>If this bundle's state is {@code ACTIVE} then this method returns
146      * immediately.
147      ´* <li>If the {@link #START_TRANSIENT} option is not set then set this
148      * bundle's autostart setting to <em>Started with declared activation</em>
149      * if the {@link #START_ACTIVATION_POLICY} option is set or
150      * <em>Started with eager activation</em> if not set. When the Framework is
151      * restarted and this bundle's autostart setting is not <em>Stopped</em>,
152      * this bundle must be automatically started.
153      * <li>If this bundle's state is not {@code RESOLVED}, an attempt is made to
154      * resolve this bundle. If the Framework cannot resolve this bundle, a
155      * {@code BundleException} is thrown.
156      * <li>If the {@link #START_ACTIVATION_POLICY} option is set and this
157      * bundle's declared activation policy is {@link org.osgi.framework.Constants#ACTIVATION_LAZY
158      * lazy} then:
159      * <ul>
160      * <li>If this bundle's state is {@code STARTING} then this method returns
161      * immediately.
162      * <li>This bundle's state is set to {@code STARTING}.
163      * <li>A bundle event of type {@link org.osgi.framework.BundleEvent#LAZY_ACTIVATION} is fired.
164      * <li>This method returns immediately and the remaining steps will be
165      * followed when this bundle's activation is later triggered.
166      * </ul>
167      * <li>This bundle's state is set to {@code STARTING}.
168      * <li>A bundle event of type {@link org.osgi.framework.BundleEvent#STARTING} is fired.
169      * <li>The {@link org.osgi.framework.BundleActivator#start} method of this bundle's
170      * {@code BundleActivator}, if one is specified, is called. If the
171      * {@code BundleActivator} is invalid or throws an exception then:
172      * <ul>
173      * <li>This bundle's state is set to {@code STOPPING}.
174      * <li>A bundle event of type {@link org.osgi.framework.BundleEvent#STOPPING} is fired.
175      * <li>Any services registered by this bundle must be unregistered.
176      * <li>Any services used by this bundle must be released.
177      * <li>Any listeners registered by this bundle must be removed.
178      * <li>This bundle's state is set to {@code RESOLVED}.
179      * <li>A bundle event of type {@link org.osgi.framework.BundleEvent#STOPPED} is fired.
180      * <li>A {@code BundleException} is then thrown.
181      * </ul>
182      * <li>If this bundle's state is {@code UNINSTALLED}, because this bundle
183      * was uninstalled while the {@code BundleActivator.start} method was
184      * running, a {@code BundleException} is thrown.
185      * <li>This bundle's state is set to {@code ACTIVE}.
186      * <li>A bundle event of type {@link org.osgi.framework.BundleEvent#STARTED} is fired.
187      * </ol>
188      * <b>Preconditions </b>
189      * <ul>
190      * <li>{@code getState()} in &#x007B; {@code INSTALLED}, {@code RESOLVED}
191      * &#x007D; or &#x007B; {@code INSTALLED}, {@code RESOLVED},
192      * {@code STARTING} &#x007D; if this bundle has a lazy activation policy.
193      * </ul>
194      * <b>Postconditions, no exceptions thrown </b>
195      * <ul>
196      * <li>Bundle autostart setting is modified unless the
197      * {@link #START_TRANSIENT} option was set.
198      * <li>{@code getState()} in &#x007B; {@code ACTIVE} &#x007D; unless the
199      * lazy activation policy was used.
200      * <li>{@code BundleActivator.start()} has been called and did not throw an
201      * exception unless the lazy activation policy was used.
202      * </ul>
203      * <b>Postconditions, when an exception is thrown </b>
204      * <ul>
205      * <li>Depending on when the exception occurred, bundle autostart setting is
206      * modified unless the {@link #START_TRANSIENT} option was set.
207      * <li>{@code getState()} not in &#x007B; {@code STARTING}, {@code ACTIVE}
208      * &#x007D;.
209      * </ul>
210      *
211      * @param options The options for starting this bundle. See
212      *                {@link #START_TRANSIENT} and {@link #START_ACTIVATION_POLICY}. The

214      * @throws org.osgi.framework.BundleException
215      *                               If this bundle could not be started.
216      *                               BundleException types thrown by this method include:
217      *                               {@link org.osgi.framework.BundleException#START_TRANSIENT_ERROR},

485      * {@link org.osgi.framework.wiring.FrameworkWiring#refreshBundles(java.util.Collection, org.osgi.framework.FrameworkListener...)

491      * {@code IllegalStateException} is thrown.
492      * <li>If this bundle's state is {@code ACTIVE}, {@code STARTING} or
493      * {@code STOPPING}, this bundle is stopped as described in the
494      * {@code Bundle.stop} method. If {@code Bundle.stop} throws an exception, a
495      * Framework event of type {@link org.osgi.framework.FrameworkEvent#ERROR} is fired containing
496      * the exception.
497      * <li>This bundle's state is set to {@code UNINSTALLED}.
498      * <li>A bundle event of type {@link org.osgi.framework.BundleEvent#UNINSTALLED} is fired.
499      * <li>This bundle and any persistent storage area provided for this bundle
500      * by the Framework are removed.
501      * </ol>
502      * <b>Preconditions </b>
503      * <ul>
504      * <li>{@code getState()} not in &#x007B; {@code UNINSTALLED} &#x007D;.
505      * </ul>
506      * <b>Postconditions, no exceptions thrown </b>
507      * <ul>
508      * <li>{@code getState()} in &#x007B; {@code UNINSTALLED} &#x007D;.
509      * <li>This bundle has been uninstalled.
510      * </ul>
511      * <b>Postconditions, when an exception is thrown </b>
512      * <ul>
513      * <li>{@code getState()} not in &#x007B; {@code UNINSTALLED} &#x007D;.
514      * <li>This Bundle has not been uninstalled.
515      * </ul>
516      *
517      * @throws org.osgi.framework.BundleException
